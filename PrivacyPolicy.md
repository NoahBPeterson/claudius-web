# Privacy Policy — Claudius Mascot

_Last updated: June 20, 2026_

**Claudius Mascot collects no data. None.**

Claudius Mascot is a browser extension that draws a small decorative animated
character on top of the message box at [claude.ai](https://claude.ai). That is its
only function.

## What we collect

Nothing. Specifically, the extension:

- **does not collect, store, or transmit any personal or usage data;**
- **makes no network requests** — it only loads an image file bundled inside the
  extension itself;
- **does not use** cookies, local storage, analytics, trackers, or third-party
  services;
- **does not read, record, or send** anything you type, view, or do on claude.ai
  or any other site;
- **does not have** a backend server — there is nowhere for data to go.

## Permissions

The extension runs a single content script on `claude.ai` solely to position and
animate the on-screen mascot. It requests no other browser permissions and no
access to your accounts, history, or files.

## Changes

If this ever changes, this document will be updated and the effective date above
revised. Material changes would be reflected in a new extension version with
updated store disclosures.
